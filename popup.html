Auto Scroller

Keyboard Shortcuts
Ctrl + Shift + S -> Start / Stop scrolling
Ctrl + Shift + up -> Increase speed
Ctrl + Shift + down -> Decrease speed

Notes:
-> Scroll duration controls full page scroll time
-> Timer is optional
-> Auto-stops at page bottom
-> Works on static pages
-> After "Scroll Time" screen will stop scrolling
-> For shortucts to work, add it here: chrome://extensions/shortcuts
